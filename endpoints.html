API Endpoints

All available REST endpoints. Base URL: https://botliy.online

Base URL https://botliy.online

Post Stats

POST /api/bots/:id/stats

Post your bot's current server and shard count to botliy. Call this on a schedule — every 15–30 minutes is ideal. Auth Required

Path Parameters
ParamTypeDescription
id required string Your bot's Discord snowflake ID.
Request Body
FieldTypeDescription
server_count required integer Total number of servers your bot is in.
shard_count integer Total number of shards. Omit if not sharding.
Example Request Body
JSON
{ "server_count": 1450, "shard_count": 2 }
Response Codes
200Stats updated successfully.
401Missing or invalid Authorization token.
404Bot ID not found or not approved.
429Rate limit exceeded.
Get endpoints

Get Stats

GET /api/bots/:id/stats

Retrieve a bot's registered server count and shard count from botliy. Public

Response
JSON
{ "error": false, "server_count": 1450, "shard_count": 2 }

Get Votes

GET /api/bots/:id/votes

Returns a list of the 1,000 most recent votes for your bot, sorted by most recent first. Auth Required

Response
JSON
{ "error": false, "votes": [ { "username": "Snowy", "id": "123456789012345678", "date": "2026-02-26T12:00:00.000Z" } ] }
Response Codes
200Returns the vote list array.
401Missing or invalid Authorization token.

Check User Vote

GET /api/bots/:id/check

Check whether a specific user has voted for your bot in the last 12 hours. Auth Required

Query Parameters
Query ParamTypeDescription
userId required string The Discord snowflake ID of the user to check.
Example Request
URL
GET /api/bots/123456789012345678/check?userId=876543210987654321
Response
JSON
{ "error": false, "voted": 1 // 1 = voted in last 12h, 0 = not voted }
